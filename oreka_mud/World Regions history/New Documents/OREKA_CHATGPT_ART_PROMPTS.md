## HOW SIZES WORK IN THE LAYOUT TOOL

The tool uses these orientations depending on zone:

| Zone          | Best image shape | Ratio  |
|---------------|-----------------|--------|
| Left / Right  | PORTRAIT        | 2 : 3  |
| Top / Bottom  | LANDSCAPE       | 3 : 2  |
| Corner zones  | SQUARE          | 1 : 1  |
| Full page     | PORTRAIT        | 2 : 3  |
| Template layer| PORTRAIT        | 2 : 3  |

Ask ChatGPT for these exact pixel sizes:
- Portrait:  896 × 1344
- Landscape: 1344 × 896
- Square:    1024 × 1024

## ═══════════════════════════════════════
## TIPS FOR GETTING CONSISTENT RESULTS
## ═══════════════════════════════════════

1. Always request the exact pixel size in your ChatGPT prompt:
   "Generate this at 896x1344 pixels" or "1344x896 pixels"

2. If ChatGPT adds its own border/frame to the art, add this line:
   "No decorative border. No frame. Raw illustration only."

3. If the image comes out too bright/modern, add:
   "Slightly desaturated. Aged. Printed look, not digital."

4. If the character is too small in frame, add:
   "Character fills 80% of the frame height."

5. Save all files as PNG before uploading to the layout tool.

6. The layout tool will warn you if DPI is below 300.
   ChatGPT images are 72 DPI by default — run them through
   an upscaler (Upscayl is free) before layout for print quality.
